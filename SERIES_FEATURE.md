# 시리즈 기능 사용 가이드

## 개요
블로그 포스트에 시리즈(연재) 기능이 추가되었습니다. 같은 시리즈의 포스트들을 자동으로 묶어서 네비게이션을 제공합니다.

## 사용 방법

### 1. 포스트에 시리즈 추가
MDX 파일의 frontmatter에 `series` 필드를 추가하세요:

```yaml
---
title: "시리즈의 첫 번째 글"
date: 2026-03-17
description: "설명"
categories: ["카테고리"]
tags: ["태그"]
series: "시리즈 이름"
---
```

### 2. 여러 포스트를 같은 시리즈로 묶기
같은 `series` 값을 가진 포스트들이 자동으로 연결됩니다:

```yaml
# 첫 번째 포스트 (2026-03-16)
series: "블로그 구축 일지"

# 두 번째 포스트 (2026-03-17)
series: "블로그 구축 일지"

# 세 번째 포스트 (2026-03-18)
series: "블로그 구축 일지"
```

## 기능 특징

- **자동 정렬**: 날짜순으로 오래된 것부터 표시 (1편, 2편, 3편...)
- **현재 글 하이라이트**: 현재 보고 있는 포스트가 강조 표시됨
- **접이식 UI**: 시리즈 목록을 펼치거나 접을 수 있음
- **조건부 표시**: 시리즈에 2개 이상의 포스트가 있을 때만 표시

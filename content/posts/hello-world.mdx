## 새로운 시작

기존 Jekyll 기반 블로그를 Next.js 16으로 완전히 새롭게 구축했습니다.

### 왜 이전했나?

- **자유로운 커스터마이징**: 테마, 레이아웃, 검색 등 모든 요소를 직접 제어
- **최신 기술 스택**: Next.js 16, Tailwind CSS v4.2, TypeScript
- **더 나은 성능**: SSG + 이미지 최적화 + CDN

### 코드 하이라이팅 테스트

```typescript
function greet(name: string): string {
  return `안녕하세요, ${name}님!`;
}

console.log(greet("이영수"));
```
